2-D Array

Source: Blue Pelican Java by Charles E. Cook

Initializing 2-D Array

Like its name suggests, 2-D arrays are just like arrays, but with both rows and columns. It can be initialized in three ways:
int score[][] = new int[3][2]; //The first bracket indicates the number of rows and the second bracket indicates the number of columns.
int score[][] = { {1,2}, {3,6}}; //Initialization and declaration on the same line
int score[][] = new int[][] {{2,4}, {6,7}}; //Same as the second method but with extra words.

Rows and Columns

Rows come first and goes horizontally. Columns come second and goes vertically. The length of a row(the amount of columns) can be determined the follwing way:
score[0].length //Returns the length of the first row in score

Ragged Array

Not all rows in a 2-D array have to have the same amount of column. When the amount of column is not the same, the 2-D array is called a ragged array. Ragged arrays are created like this:
int a[][] = new int[2][] //# of rows declared but not column
a[0] = new int [4]; //row 0 has 4 columns
a[1] = new int [3]; //Row 1 has 3 columns
a[2] = new int [1]; // Row 2 has 1 column
