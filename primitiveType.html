Primitive Types

Source: Blue Pelican Java by Charles E. Cook

All Java primitive types:

  • byte
  • char
  • short
  • int
  • long
  • float
  • double
  • boolean

Declaring and initializing:

We can declare and initialize at the same time like this:
double x = 2;
We are declaring x to be of type double and we are initializing x to the value of 2. All this can also be done in two lines of code (as shown below) instead of one if desired:
double x; //this declares x to be of type double
x = 1.6; //this initializes x to a value of 1.6

Rules & Conventions

  • Value assigned to a variable must match the variable type
  • Declaration must come before initialization
  • Start your variable name with a lowercase letter
  • Separate words in variable name with "_" or with camel case
