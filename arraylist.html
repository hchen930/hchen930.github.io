ArrayList

Source: Blue Pelican Java by Charles E. Cook

Why ArrayList

Arraylist serves the same purpose as arrays: store multiple variables of the same type in one place. However, unlike arrays, the size of arraylists are flexible. Values can be added to expand the list and values can be deleted to shrink the list. We no longer have to make an array super large when we're not sure about what size we need. However, arraylist only stores objects so any primitive types has to be converted to objects using wrapper class.

Creating an ArrayList

Arraylist class implements the List interface so an arraylist can be created in two ways:
List scores = new ArrayList<>();
or
ArrayList = new ArrayList<>();
