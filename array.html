Creating an Array

An array in Java is used to store multiple variables of the same type in one place. There are three ways to declare and initialize arrays:

  1. //Here, the size of the array is stated within the brakcets and cannot be changed
    int arrayName[] = new int[300];
  2. //In the second way, the values are inputed directly during while initializing and size will be determined by how many elements are present.
    int arrayName[] = {23,35,676};
  3. //The third way is almost the same as the second way
    int arrayName[] = new int[] {23,35,676};

Accessing and Changing Values in Array

To access a value stored in an array, we have to specify its index. Note that index of array starts at 0 instead of 1.
//Here, the value at the fourth container in arr is stored in value
int value = arr[3];
Since the length of arrays can't be changed, we can only modify existing variables in the array and not add new ones.
arr[3] = 6; //change the value stored in array index 3 to 6
