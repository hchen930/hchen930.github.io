Objects

Source: Blue Pelican Java by Charles E. Cook

Instantiating an object

When creating an object, use the following format:
className variableName = new className(parameter, parameter);
Some classes also have a default constructor where you don't have to pass in parameters because the default ones are set for you.

Referencing an object's field

When accessing an object's field variables outside its class, the dot operator must be used along with the object's name. For example:
variable = objectName.name;
Alternatively, you can "new" to create a new object from a class and access its field:
variable = new className().height;

Calling an object's methods

When calling a method, we also use the dot operator with the object name and method name:
variable = objectName.getName(argument, argument);
