Creating A Class

A class in Java is a template that are used to create an object. It contains all the properties of an object such as its methods, fields, and data type. The structure shown below is an example of a class.
public class className {
private int fieldName; //this is a field
public className (parameters) {} //note that the constructor always shares the same name as the class name.
public int methodName() {} //This is a method
}

Parts to a Method Signature

Here's a sample method:
public int getGrade(String name, int class){}

  1. First of all, we have to declare the access control of the method. The public keyword here makes the method accessible outside of this class. If we write private instead, this method will only be accessible inside class of this method.
  2. The next keyword is int. This is the type of value this method will return if it's called. If you don't want to return anything, write void.
  3. Next, we have the name of this method. Here, the method name is getGrade.
  4. Next, we have the set of parameter needed to call this method. We include the type of value and a temporary name that will be used inside this method only. If there's more than one parameter, separate them with commas. If there's no parameters needed, still include an empty set of empty parenthesis.

The Main Method

public static void main (String args[]) {}
In this main method that we have to write each time for the code to run, the main here means that this is where the entry point is when Java interprets your code.
