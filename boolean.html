Boolean Expression & If Statements

Source: Blue Pelican Java by Charles E. Cook

List of boolean expressions (in order)

  1. ! (not)
  2. == (equals)
  3. != (not equal to)
  4. && (and)
  5. || (or)

When multiple boolean expressions are in the same line, Java will execute them in the precedence listed above. Alternatively, you can always use parenthesis to give first precedence.

a b a&&b
false false false
false true false
true true true
a b a||b
false false false
false true true
true true true

Using if statements

If statements forms the logic of java. Here's the basic structure of if statements:
if (boolean expressions) {code}
If the boolean expression is true, then the code will be executed. If not, there are more if statement structure available.
if (boolean expression) {code} else {alternative code}
if the boolean expression is false in this one, the alternative code is executed.
if (boolean expression){code} else if (alternative boolean expression){alternative code}
Here, the alternative code will excecute is orginal expression is false and alternative expression is true.
